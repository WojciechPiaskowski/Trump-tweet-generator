The project was an interesting learning experience, trying different approaches on word2vec, representing the words in multi-dimensional space,
different combinations of neural layers.

However GAN models are better and generating pictures than text, as a small deviation from target might change the sentance completely, while
pixel that has a color similar to target but not exact will give better results.

Another issue was twitter speciffic issues connected to posting links, tagging event and people using @ and nicknames.

Another approach for text generation could be LSTM models for sequence prediction, which I hope I will have time to try in the future

The file with google word2vec network was removed from git because of size of the file. You can find it here:
https://code.google.com/archive/p/word2vec/
